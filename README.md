# Spendora – Personal Finance Dashboard

A modern, responsive Personal Finance Dashboard built using **HTML, CSS, and JavaScript**.

Track income, expenses, set monthly budgets, and visualize spending insights with interactive charts.

##  Features
-  Real-time Expense Breakdown Chart (Chart.js)
-  Income & Expense Tracking
-  Monthly Budget Limit with Progress Bar
-  Search, Filter & Sort Transactions
-  Dark / Light Mode Toggle
-  LocalStorage Data Persistence
-  Fully Responsive Layout
-  
## Tech Stack
- HTML5
- CSS3 (Flexbox + Responsive Design)
- JavaScript (ES6)
- Chart.js
